# overwatch-lucky-wheel

## 1. 개요
  - 기존에 만들었던 `/lucky-wheel` 프로젝트를 기반으로, 오버워치 전용 룰렛을 만드는 프로젝트다.
  - 기존에 만들었던 `/lucky-wheel` 프로젝트의 코드를 클론해서, 프로젝트 루트폴더 이름을 overwatch-lucky-wheel로 바꾸고, git 기록을 삭제한 다음 다시 첫 커밋을 단 상태로 시작한다.
  - 기존 코드에 들어있는 `lucky-wheel` 프로젝트 이름을 `overwatch-lucky-wheel`로 바꾸어야 한다.

## 2. 기술스택
  - 현재 프로젝트가 단순한 HTML+CSS+Javascript 기반인 것을 확인했다.
  - 이걸 React+TailwindCSS 기반으로 전환할 예정이다.

## 3. 들어갈 기능
  - 오버워치 역할군 중 어떤 걸 고를지 선택하는 뽑기유형이 있다.
  - 오버워치 전체영웅 중 어떤 영웅을 고를지 선택하는 뽑기유형이 있다.
  - 오버워치 역할군 내 영웅 중 어떤 영웅을 고를지 선택하는 뽑기유형이 있다.
  - 각 뽑기유형에는 기본으로 각 뽑기유형에 해당하는 모든 영웅이 자동으로 항목으로 들어가되, 사용자가 직접 항목을 추가하거나 제거할 수 있는 기능이 있어야 한다.
  - 각 뽑기유형에는 각 항목별 당첨갯수를 지정하거나, 당첨확률을 정할 수 있어야 한다.
  - 사용자가 항목과 확률값을 입력한 후 완료 버튼을 누르면 항목이 저장되고, 추첨 버튼을 클릭하면 돌림판이 돌아가다가 특정 항목을 추첨할 수 있도록 한다.
  - 돌림판이 정지하고 나면 당첨된 항목을 그 확률과 함께 나타내고, 다시 추첨할 수 있도록 하거나 항목을 수정할 수 있도록 한다.
  - 전체적인 흐름은 기존의 코드 베이스와 비슷한 느낌으로 진행한다.


# Lucky Wheel - 당첨룰렛

## 1. 개요
  - 사용자가 직접 추첨에 들어갈 항목과 그 확률을 정할 수 있는 룰렛이다.
  - 이 프로젝트는 javascript만을 사용, 추첨항목과 확률을 정해 돌아가는 돌림판을 구현하는 것이 그 목적이다.
  - 향후 React로 프론트엔드를 적용할 계획이나, 일단은 javascript로 필수 기능을 구현하는 것이 주 목적이다.

## 2. 들어갈 기능
  - 사용자가 직접 당첨에 들어가는 항목을 정할 수 있어야 한다.
  - 사용자가 각 항목별 당첨확률을 정할 수 있어야 한다.(확률별, 당첨갯수별)
  - 사용자가 각 항목과 확률값을 입력한 후 완료 버튼을 누르면 항목이 저장되고, 추첨 버튼을 클릭하면 돌림판이 돌아가다가 특정 항목을 추첨할 수 있도록 한다.
  - 돌림판이 정지하고 나면 당첨된 항목을 (그 확률과 함께) 나타내고, 다시 추첨을 할 수 있도록 하거나 항목을 수정할 수 있도록 한다.

## 3. 기술스택
  - 기본뼈대: HTML
  - 스타일링: CSS + TailwindCSS
  - 기능구현: Javascript (향후 React 업데이트 가능성)
